Windows 下 Emacs 中的 zsh shell

2019-03-12
分类： 技术
标签： shell zsh

运行跨平台 shell（例如 Bash 或 zsh）的最大优势在于你能在多平台上使用同样的语法和脚本。在 Windows 上设置（替换）shell 挺麻烦的，但所获得的回报远远超出这小小的付出。
